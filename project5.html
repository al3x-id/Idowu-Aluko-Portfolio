SuperStore Sales Data Cleaning - Python Project

Project Overview

There are two batches of superstore sales data from different teams. The files are similar but messy and inconstitent. The files include two sheets Batch1(first 3500 rows) Batch 2(second 3500 rows) which must be appended together before or after cleaning.

Objectives

1. Append the Data

  • Combine the rows in Batch 1 and Batch 2 into a single sheet called Combined Data.

2. Split "Customer Info"

  • From the Customer Info column, create: Customer Name & Customer ID

3. Clean "order Date

  • Convert all values into a proper date format.
  • Replace "invalid date" with a blank or a fixed default date (e.g., 01/01/2021) and mark the row for review.

4. Standardize "Ship Mode"

  • Correct inconsistent values such as standard class, Same day, blank, etc.
  • Use Standard Class, Second Class, Same Day, First Class only.

5. Clean the "State" Column

  • Fix variations:
  • calif. → California
  • NY → New York
  • Empty or NaN

6. Fix "Category" Column

  • Normalize values to: Technology, Furniture, Office Supplies
  • Replace incorrect ones like Furn, Office supply, or blanks.

7. CleAN "Sales" Column

  • Remove currency signs, commas, and text values: $300 → 300 & 125,00 → 125.00
  • Replace blanks and NaN with 0 and flag for checking.
  • Add a standard format

8. Fix "Quantity" Column

  • Convert everything to numbers: "five" → 5 & Blank, NaN → replace with 0 or highlight for manual review.

Questions

  • What is the Total Sales Value?
  • How many customers used the Standard Class ship mode?
  • How many unique customers are in the dataset?

2. Append the Two Datasets as combined_data Using ".concat()"


combined_data = pd.concat([df1,df2])

Reset Index to Append the Dataset Index as One Using ".reset_index()"


combined_data.reset_index(drop = True, inplace = True)

3. Split the Customer Info Column into Two Separate Columns (Customer Name & Customer ID) Using ".split()" 


combined_data[['Customer Name', 'CustomerID']] = combined_data['Customer Info'].str.split('|', expand = True)

4. Delete the Customer Info Column Using ".drop()"


combined_data = combined_data.drop(columns = 'Customer Info')

Moved Customer Name and CustomerID Columns to 2nd and 3rd position with ".iloc()"


combined_data = combined_data.iloc[:, [0,8,9,1,2,3,4,5,6,7]]

5. Convert all Values in Order Date to Proper Date Format and Fill the Missing Date with the Default Date `2021-01-01 Using ".to_datetime()" & ".fillna"


combined_data['Order Date'] = pd.to_datetime (combined_data['Order Date'], errors = 'coerce').fillna('2021-01-01')

6. Fill Missing Rows in Ship Mode and State with Unknown Using ".fillna()"


combined_data[['Ship Mode','State']] = combined_data[['Ship Mode', 'State']].fillna('Unknown')

Standardize the Ship Mode Column Using ".title()"


combined_data['Ship Mode'] = combined_data['Ship Mode'].str.title()

7. Cleaned the `State` column to Fix Variations (calif. → California, NY → New York, TX → Texas) Using ".replace()" 


combined_data['State'] = combined_data['State'].replace(['calif.', 'NY','TX'], ['California', 'New York', 'Texas'])

8. Normalize Values to: Technology, Furniture, Office, Office Supplies. Using Logical Rules to Assign Consistent Category to each Product Using "for loop" with "if" Statement


for x in combined_data.index:
    if combined_data.loc[x, 'Product Name'] in ('Office Chair', 'Desk'):
        combined_data.loc[x, 'Category'] = 'Furniture'
    elif combined_data.loc[x, 'Product Name'] in ('Monitor', 'Wireless Mouse', 'Laptop'):
        combined_data.loc[x, 'Category'] = 'Technology'
    else:
        combined_data.loc[x, 'Category'] = 'Office supplies'

9. Fill Missing Rows in Sales and Quantity Column with '0' Using ".fillna()"


combined_data[['Sales', 'Quantity']] = combined_data[['Sales', 'Quantity']].fillna(0)

Removed Currency, Comma and Text Values from Sales COlumn Using ".replace()" & ".astype()" to Change the Data Type to Float


combined_data['Sales'] = combined_data['Sales'].replace(['$300','125,00'],['300.00', '125.00']).astype(float)

10. Replaced Five with '5' Using ".replace() and Convert the Quantity Column to Integer Using ".astype()"


combined_data['Quantity'] = combined_data['Quantity'].replace('five', 5).astype(int)
